# Atelier MCP - Slides de présentation

Présentation interactive pour un atelier de 45 minutes sur les **MCP (Model Context Protocol)**.

**[Voir la présentation en ligne](https://benoitvx.github.io/atelier-mcp-slides/)**

## Contexte

- **Public** : Agents publics non-techniques (réseau ALLiaNCE)
- **Présentateur** : Benoit Vinceneux, EIG CPO Données & MCP @ Etalab/DINUM
- **Durée** : 45 minutes

## Contenu

27 slides organisées en 6 sections :

1. **Introduction** (3 slides) - Présentation et objectifs
2. **Ice breaker** (1 slide) - Sondage interactif
3. **Pédagogie MCP** (11 slides) - Concepts, architecture, comparaisons
4. **Exemples concrets** (7 slides) - Démos et cas d'usage dans l'État
5. **Usages** (3 slides) - Checklist et ressources
6. **Conclusion** (2 slides) - Questions et contacts

## Stack technique

- React 18 + TypeScript + Vite
- [@codegouvfr/react-dsfr](https://react-dsfr.codegouv.studio/) - Design System de l'État français
- CSS Modules
